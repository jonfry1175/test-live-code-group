# React + Vite

* install pnpm 

```bash
npm install -g pnpm
```

```bash
cd client
```

* install package

```bash
pnpm i
```

* run
```bash
pnpm run dev
```

## Cara Menggunakan API

1. pilih sesuai dengan sistem operasi yang dimiliki.
2. Jalankan API menggunakan terminal dengan perintah sebagai berikut:
   ```bash
    ./enigma-laundry-app-macos
   ```
